关于IDEA2017配置web服务器注意事项

发表于 2021-08-03 阅读次数：

第一，Add Framework Support

（1）鼠标放在创建的module 项目上，右击选择Add Framework Support ，勾选web application。

（2）会发现，少了WEB-INF，点击Project structure ，选择Facets-Web项目-右边的 “ + “ 号，加入WEB-INF 。Apply and close

（3）把WEB-INF拖到web目录下。

第二，加入lib文件夹

（1）点击Project structure，选择Artifacts（打包），查看右边的WEB-INF 下是否lib 包，没有新建一个文件夹(create directory)，名字是lib

（2）创建了lib 之后，选中lib文件夹，点击 “+” 号，Library files，把所有的文件选中，然后Apply
